## Command Explanation

* Repeat :   重复内部操作，输入为次数
* Light > :    当灰度值大于输入时执行内部操作
* Distance > :    当距离大于输入时执行内部操作
* Light < :    当灰度值小于输入时执行内部操作
* Distance < :    当距离小于输入时执行内部操作
* Left :    向左转，输入为角度
* Right :    向右转，输入为角度
* Forward :    向前进，输入为距离，单位cm
* Circle :    转圈，输入表示旋转方向
* Dance :    跳舞，输入表示持续时间
* SelectGait :    步态调整，有1，2，3三种选择
